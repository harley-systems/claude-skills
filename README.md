# Claude Skills - Harley Systems

Claude Code skills marketplace for Harley Systems tools.

## Installation

```
/plugin marketplace add harley-systems/claude-skills
```

Then use `/plugin` to install individual skills.

## Available Skills

### sca-plugin

SCA (Simple Certificate Authority) CLI assistant. Provides command reference, YubiKey/PKCS#11 integration docs, workflow guides, and development documentation for the [SCA](https://github.com/harley-systems/sca) tool.

### openwrt-plugin

Ansible collection [harley.openwrt](https://github.com/harley-systems/ansible-collection-openwrt) assistant. Covers all 10 roles (base, network, firewall, dhcp_dns, ipsec, ipsec_gateway, acme, ddns, ddns_gateway, webui), the UCI module, inventory conventions, playbook patterns, and troubleshooting for OpenWRT router and Linux gateway configuration via Ansible.
